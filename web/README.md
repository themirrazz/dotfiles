# Webrices
## Styles
* **GoogleDocs.css:** Add some fun effects to Docs, Sheets, and Slides
* **IAintBuyingShit.css:** Remove the Gemini and Upgrade button in Google Docs
## TamperMonkey
* **googleDocsMacBar.js:** Zoom effect when you hover the toolbar in Docs, Sheets, and Slides
* **googleSlidesPresentAnimation.js:** Makes a curtain open and close when you press the "Present" button in Google Slides (requires `GoogleDocs.css`)
